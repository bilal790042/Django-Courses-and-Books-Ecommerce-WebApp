It is a project where we build  a platform using Django and React that have the functionality of offering books and courses services in a single place. where you are able to authenticate, upload and manages books and
courses.

unique feature of this project is that it offers one to one mentoring session where student can consult teacher enrolled in his course can ask and have a chat with teacher where he provides information relevent to the 
query.
